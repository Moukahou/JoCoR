# JoCoR/Co-teaching

## Enviornments

 - python 3.7
 - PyTorch 1.3.1
 - torchvision 0.4.2
 - Numpy
 
## How to run

```
git clone https://github.com/Moukahou/JoCoR.git
```
and then 
```
cd JoCoR
python train.py
```

you can adjust the noise rate of the training data and some other parameters by modifying the config.py
